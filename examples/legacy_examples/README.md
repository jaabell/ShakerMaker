# legacy_examples

Original ShakerMaker examples from the upstream repository (José A. Abell),
kept here **unmodified** as a regression reference.

They all use the classic `ShakerMaker.run()` path (not the OP pipeline), so if
any of them stops working after a change, the original workflow was broken.

These are reference material only — the maintained, structured examples live in
the numbered folders (`01_crustmodel/`, `02_sources/`, ...).
